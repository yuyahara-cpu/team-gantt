ログイン

チーム・タスクガント（WP課）

アカウントは管理者（原）が発行します。

初期設定：Firebase（無料）に接続してください

このアプリは全員でリアルタイム共有するために、無料のFirebase(Firestore)を使います。下記を一度だけ設定すれば、以降は誰でもこのページを開くだけで共同編集できます。

  1. Firebase Console で無料プロジェクトを作成。
  2. 「Firestore Database」を作成（本番/テストどちらでも可。まずはテストモードで開始が簡単）。
  3. 「Authentication」→ Sign-in method で「メール/パスワード」を有効化し、「Users」タブでメンバー分のアカウント（メール＋パスワード）を追加。
  4. プロジェクト設定 → 「マイアプリ」でWebアプリを追加し、表示される firebaseConfig をコピー。
  5. この app.html の先頭 FIREBASE_CONFIG に貼り付けて保存 → GitHub等に再アップロード。

Firestoreのセキュリティルール例（匿名ログイン済みのみ読み書き可）：

rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /tasks/{id} {
      allow read, write: if request.auth != null;
    }
  }
}
